	13. Vue Cli로 뷰 설치하기
	- Vue Command Line Interface
	- Vue와 관련된 package를 설치할 때 간단한 명령어로 할 수 있게 도와주는 도구
	- Vue Cli는 노드가 설치되어 있어야 한다.

	- npm install -g @vue/cli
	- vue --version

---------------------------------------------------------------------------------
	- npx @vue/cli create test2


	* node_modules: npm package들이 들어간다.
	* package/json: package dependencies와 개발 dependencies
	* src: 개발하면서 코딩을 할때 소스 코드들이 들어간다.

	* npm run serve
